# 🤝 Collaboration

Vous souhaitez contribuer à docsDirecte ? C'est grâce à la communauté que cette documentation reste à jour et complète !

## Pourquoi collaborer ?
L'API d'EcoleDirecte change souvent, et nous ne pouvons pas tout voir seuls.
En apportant votre aide, vous aidez des centaines d'autres développeurs et vous serez ajouté à la page des [Crédits](/credits).

## Comment aider ?

Voici les étapes recommandées :

### 1. 🔍 Recherches
Explorez l'API par vous-même en utilisant les outils de développement de votre navigateur sur EcoleDirecte. Notez les points de terminaison (endpoints), les paramètres envoyés et la structure des réponses. Nous utilisons Firefox DevTools et HTTP Toolkit pour cela.

### 2. 📝 Formulation des recherches
Structurez vos découvertes de manière claire :
- Quel est l'URL de l'endpoint ?
- Quelle est la méthode HTTP (GET, POST, etc.) ?
- Quel est le format de la requête et de la réponse ?
- Quelles sont les conditions nécessaires (ex: type de compte, token) ?

### 3. 🚀 Passer à l'action
Une fois vos recherches prêtes, vous avez deux options :

- **Créer une Issue sur GitHub :** Si vous avez trouvé une erreur ou une nouvelle info mais ne savez pas comment modifier la doc.
- **Créer une Pull Request :** Si vous êtes prêt à modifier le code ou le texte directement.
